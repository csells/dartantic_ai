# Logging

Control debug output and monitoring with Dartantic's logging system.

## Quick Start

```dart
// Enable default logging
Agent.loggingOptions = const LoggingOptions();

final agent = Agent('openai');
await agent.send('Hello!');
// See formatted logs in console
```

## Logging Levels

```dart
import 'package:logging/logging.dart';

// Development - verbose output
Agent.loggingOptions = LoggingOptions(level: Level.FINE);

// Production - minimal output  
Agent.loggingOptions = LoggingOptions(level: Level.WARNING);
```

| Level | Use Case | Output |
|-------|----------|--------|
| `Level.FINE` | Development | Verbose |
| `Level.INFO` | General | Moderate |
| `Level.WARNING` | Production | Minimal |

## Filter by Provider

```dart
// Only OpenAI logs
Agent.loggingOptions = LoggingOptions(
  filter: 'openai',
  onRecord: (record) => print('OpenAI: ${record.message}'),
);

// Only HTTP retry logs
Agent.loggingOptions = LoggingOptions(filter: 'http');
```

## Custom Output

### Colored Console
```dart
Agent.loggingOptions = LoggingOptions(
  onRecord: (record) {
    final color = record.level == Level.WARNING ? '\x1B[33m' : '';
    print('$color[${record.level.name}] ${record.message}\x1B[0m');
  },
);
```

### JSON Logging
```dart
Agent.loggingOptions = LoggingOptions(
  onRecord: (record) => print(jsonEncode({
    'timestamp': DateTime.now().toIso8601String(),
    'level': record.level.name,
    'message': record.message,
  })),
);
```

## Logger Hierarchy

```
dartantic
├── agent         # Agent operations
├── chat
│   ├── openai    # OpenAI provider
│   ├── anthropic # Anthropic provider
│   └── google    # Google provider
├── http.retry    # Rate limiting
└── tools         # Tool execution
```

## Common Patterns

### Debug Rate Limiting
```dart
Agent.loggingOptions = LoggingOptions(filter: 'http');
```

### Debug Tool Calls  
```dart
Agent.loggingOptions = LoggingOptions(filter: 'tool');
```

### Provider-Specific
```dart
Agent.loggingOptions = LoggingOptions(filter: 'anthropic');
```
