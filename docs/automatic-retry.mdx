---
title: Automatic Retry
---

Built-in retry logic handles temporary rate limits and failures.

## Zero Configuration

```dart
final agent = Agent('openai');

// Automatically retries on 429 errors
final result = await agent.send('Hello!');
```

## Timing Strategy

```
Attempt 1: Immediate
Attempt 2: 1s + jitter
Attempt 3: 2s + jitter  
Attempt 4: 4s + jitter
```
