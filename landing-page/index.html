Multi-step Tool Calling

Docs →

Give your agents real-world abilities with composable tools and automatic orchestration. 

final agent = Agent('anthropic', tools: [
  weatherTool,
  temperatureConverterTool,
  currentDateTimeTool,
]);

final result = await agent.send(
  "What's the weather in Paris? Convert to Fahrenheit and tell me the time."
);

Streaming Output

Docs →

Render responses as they happen and build real-time experiences without extra plumbing. 

await for (final chunk in agent.sendStream(
  'Tell me your best joke',
)) {
  stdout.write(chunk.output);
}

Typed Output

Docs →

Lock in structured JSON with schema support and strongly typed deserialization. 

final city = await agent.sendFor<Map<String, dynamic>>(
  'The windy city in the US of A',
  outputSchema: JsonSchema.create({
    'type': 'object',
    'properties': {
      'city': {'type': 'string'},
    },
    'required': ['city'],
  }),
);
print(city.output['city']);

Embeddings

Docs →

Generate vectors, search semantically, and build RAG-ready pipelines with a single API. 

final query = await agent.embedQuery('programming');
final docs = await agent.embedDocuments([
  'Dart language',
  'Cooking recipes',
  'Python coding',
]);

final topScore = docs.embeddings
    .map((v) => EmbeddingsModel.cosineSimilarity(
          query.embeddings,
          v,
        ))
    .reduce((a, b) => a > b ? a : b);

Thinking Metadata

Docs →

Surface model reasoning alongside final answers for debuggability and delightful UX. 

final agent = Agent(
  'openai-responses:gpt-5',
  chatModelOptions: const OpenAIResponsesChatModelOptions(
    reasoningSummary: OpenAIReasoningSummary.concise,
  ),
);

final result = await agent.send('Explain quicksort.');
stdout.write(result.metadata['thinking']);

Server-Side Tools

Docs →

Turn on provider-hosted search, file retrieval, and image generation without extra infrastructure. 

final agent = Agent(
  'openai-responses',
  chatModelOptions: const OpenAIResponsesChatModelOptions(
    serverSideTools: {OpenAIServerSideTool.webSearch},
  ),
);

await for (final chunk in agent.sendStream('Recent Dart news?')) {
  final events = chunk.metadata['web_search'];
  if (events != null) stdout.writeln(events.last['type']);
}

Provider Switching

Docs →

Mix and match OpenAI, Anthropic, Google, Mistral, Ollama, and more in the same conversation.
